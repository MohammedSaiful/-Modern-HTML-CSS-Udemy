HTML & CSS Sandbox - Block vs Inline

Block Elements

This is a div

This is a paragraph

Element Description
<div> Defines a division or container for grouping content.
<p> Represents a paragraph of text.
<h1> – <h6> Define headings from largest <h1> to smallest <h6>.
<ul> Creates an unordered (bulleted) list.
<ol> Creates an ordered (numbered) list.
<li> Defines a list item inside <ul> or <ol>.
<table> Creates a table structure.
<form> Defines an input form for user data.
<header> Represents the header section of a page or section.
<footer> Represents the footer section of a page or section.
<section> Groups related content into a thematic block.
<nav> Contains navigation links.
<article> Represents an independent piece of content (like a blog post).
<aside> Defines side content, such as a sidebar or note.
<main> Specifies the main content area of a page.
<blockquote> Displays long quotations.
<hr> Inserts a horizontal line or thematic break.
<pre> Displays preformatted text (preserves spaces and line breaks).

Inline Elements

This is a span This is a link This is bold text
Element Description
<span> Groups small pieces of text inline for styling.
<a> Creates a hyperlink to another page or resource.
<img> Embeds an image within text.
<button> Adds a clickable button.
<input> Defines an input field for user data.
<label> Associates text with a form input field.
<small> Displays text in a smaller font size.
<strong> Indicates strong importance (bold text).
<em> Emphasizes text (italic style).
<mark> Highlights text for attention.
<ins> Marks inserted text (usually underlined).
<del> Marks deleted text (usually struck through).
<sub> Displays subscript text (below baseline).
<sup> Displays superscript text (above baseline).
